# DroidNode 📱⚡

<p align="center">
  <img src="https://droidnode.ct.ws/system/logo.svg" alt="DroidNode Logo" width="200">
</p>

**DroidNode** est un serveur Web *Edge-Computing* local conçu pour la performance et la simplicité. Il vous permet de transformer instantanément votre appareil Android en un hôte de fichiers capable de servir du contenu statique (**HTML, CSS, JS**) directement sur votre réseau domestique.

---

## 🚀 Fonctionnalités

- **Edge Computing Local** : Traitez et servez vos données au plus près de l'utilisateur (votre réseau domestique).
- **Transformation Android** : Recyclez vos anciens smartphones ou utilisez votre appareil actuel comme serveur web mobile.
- **Support Web Complet** : Hébergement fluide de fichiers HTML5, feuilles de style CSS3 et scripts JavaScript.
- **Léger & Rapide** : Optimisé pour consommer un minimum de ressources sur votre appareil Android.

## 🛠️ Installation

1. Téléchargez l'application DroidNode sur votre appareil Android.
2. Placez vos fichiers web dans le répertoire dédié (ex: `/sdcard/DroidNode/www`).
3. Lancez le serveur.
4. Accédez à votre contenu via l'adresse IP affichée (ex: `http://192.168.1.15:8080`).

## 🌐 Pourquoi DroidNode ?

DroidNode répond au besoin croissant de décentralisation. Plutôt que de dépendre de serveurs cloud distants, vous utilisez la puissance de calcul déjà présente dans votre poche pour tester des sites web, partager des ressources en local ou créer des interfaces domotiques.
