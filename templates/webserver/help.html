Introduction

DroidNode est un serveur Web Edge-Computing local. Il vous permet de transformer votre appareil Android en un hôte de fichiers capable de servir du contenu HTML, CSS et JS sur votre réseau domestique.

Point technique : Le backend est propulsé par Python 3.13 et Flask, offrant une gestion asynchrone des fichiers et une sécurité renforcée contre le "Path Traversal".

Structure des fichiers

Tous les fichiers servis publiquement doivent se trouver dans :
/home/web dev/

Priorité d'affichage

Si vous visitez /web/mon-projet/, le serveur cherchera ces fichiers dans cet ordre :

  • 1 index.html
  • 2 index.htm
  • 3 index (sans extension)

Réseau & Visibilité

Votre serveur écoute sur l'interface 0.0.0.0, ce qui signifie qu'il accepte les connexions provenant de votre téléphone lui-même (localhost) et de tout appareil sur le Wi-Fi.

Adresse actuelle : http://{{ local_ip }}:8080/web

Si vous ne parvenez pas à vous connecter depuis un PC, vérifiez que l'option "Isolation AP" est désactivée sur votre routeur Wi-Fi.

Sécurité & Confidentialité

DroidNode applique plusieurs couches de protection pour protéger vos données privées dans Termux.

  • Sandboxing : Le serveur ne peut pas lire de fichiers en dehors de /home/.
  • Limitation de taille : La lecture directe de fichiers texte via l'API est limitée à 500 Ko pour éviter les plantages (RAM).
  • Sessions : L'accès au Dashboard est protégé par un cookie de session crypté.

Optimisation & Performance

Pour garantir la fluidité de votre serveur sur Android :

  • Mise en cache : Le navigateur met en cache les assets (images/CSS) par défaut.
  • Mode Sommeil : Android peut couper Termux pour économiser la batterie. Utilisez la commande termux-wake-lock pour garder le serveur actif.
  • Zippage : Lors du téléchargement de dossiers, la compression s'effectue dans /tmp/ et est nettoyée immédiatement après.

Commandes Utiles (Termux)

Voici quelques commandes à taper dans votre application Termux pour gérer l'environnement :

python app.py : Lance le serveur.

ls -la "~/web dev" : Liste vos projets web.

ifconfig : Affiche les détails de votre connexion réseau.

pkg upgrade : Met à jour les bibliothèques système.

Dépannage (Erreurs)

403 Forbidden

Ceci arrive si vous tentez de lister un dossier qui ne possède pas de fichier index.html. L'exploration de répertoire est désactivée par sécurité.

BuildError

Généralement dû à un lien incorrect dans le template Jinja2. Vérifiez vos balises url_for.

Foire Aux Questions

Puis-je utiliser PHP ?

Non, ce serveur est basé sur Flask (Python). Il sert des fichiers statiques. Pour du PHP, il faudrait installer php-apache ou nginx dans Termux.

Comment changer le port ?

Modifiez la dernière ligne de app.py : app.run(port=8080) par le port de votre choix.

Le serveur s'arrête quand je ferme Termux ?

Oui. Pour le laisser tourner en tâche de fond, ne quittez pas l'app, revenez simplement à l'écran d'accueil de votre téléphone.
